# Portfolio Web Developer — Bootstrap 5

Projekt przedstawia responsywną stronę typu **portfolio web developera** stworzoną z wykorzystaniem frameworka Bootstrap 5. Celem projektu jest nauka budowy nowoczesnych interfejsów webowych przy użyciu gotowych komponentów Bootstrapa oraz poznanie zasad tworzenia responsywnych aplikacji internetowych.

## Cele projektu

Podczas realizacji projektu można nauczyć się:

- korzystania z frameworka Bootstrap 5,
- budowania responsywnego układu strony,
- wykorzystania systemu siatki (`grid system`),
- tworzenia sekcji nawigacyjnych,
- stosowania komponentów Bootstrap,
- projektowania formularzy kontaktowych,
- organizacji struktury HTML zgodnie z dobrymi praktykami.

---

## Technologie

Projekt został wykonany z użyciem:

- HTML5
- Bootstrap 5.3.3
- CSS (klasy utility Bootstrapa)
- JavaScript Bootstrap Bundle

---

## Funkcjonalności

Strona zawiera:

- responsywne menu nawigacyjne,
- sekcję Hero z obrazem w tle,
- sekcję „O mnie”,
- sekcję umiejętności opartą o komponenty `card`,
- portfolio projektów,
- formularz kontaktowy,
- responsywny układ dostosowany do urządzeń mobilnych.

## Najważniejsze elementy Bootstrapa wykorzystane w projekcie

### Navbar

Responsywne menu nawigacyjne z przyciskiem hamburger dla urządzeń mobilnych.

### Grid System

Układ strony został zbudowany przy pomocy siatki Bootstrapa:

```html
<div class="row">
  <div class="col-md-4"></div>
  <div class="col-md-4"></div>
  <div class="col-md-4"></div>
</div>
```

### Cards

Sekcje umiejętności oraz portfolio wykorzystują komponent `card`.

### Formularz

Formularz kontaktowy wykorzystuje klasy:

```html
form-control form-label btn btn-primary
```
